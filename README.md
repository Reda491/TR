# Tenere Riders Website

Standalone React + Vite website for a motorcycle rental business.

## Run locally

1. `npm install`
2. `npm run dev`
3. Open the local URL printed by Vite

## Build

- `npm run build`
- `npm run preview`

## Notes

- This project is fully local/standalone (no Base44 dependency).
- Data is provided by local entities and a local API adapter in `src/api/client.js`.
